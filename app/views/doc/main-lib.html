Task Library

The Tatool Task Library contains the currently publically available Executables and Task Batteries and lists the Properties and other information you need to know to customise them. You can choose from single Executables and complete task batteries. Single Executables can be easily combined to be used in your modules. You can select from Experimental Tasks (customisable paradigms, typically from the cognitive, neuroscientific, and/or social psychological literature), Adaptive Training Tasks (paradigms allowing for adjusting task difficulty to individual performance), and Accessory Executables (anything else you need to run a study, such as Executables to present instructions). Task Batteries are typically associated with published scientific studies and contain ready-to-run modules including instructions and stimuli files alongside the Executables.

We're constantly working on creating new Executables that will become available for everyone. If you think you have an Executable you'd like to share with the community, let us know! Please don't forget to cite us as well as the original authors of the paradigms and the task batteries you're using.

Available Task Batteries

Name Content Authors (Year)
Everyday Problems Test (EPT-14) Three language and culture versions - English (EPT-14E; UK), French (EPT-14F; Canada), and German (EPT-14G; Germany) - of the updated and shortened Everyday Problems Test (EPT; Willis & Marsiske, 1993). Reinhartz, Strobach, Hyde, Jiang, Jacobsen, Belleville, & von Bastian (submitted for publication)
Executive Functions Three tasks for each of the executive functions inhibition (Simon, Flanker, and Stroop task), shifting (Shape/Color, Animacy/Size, and Parity/Magnitude Switching task), and updating (Color, Letter, and Number Keep-Track task). von Bastian, Souza, & Gade (2016)
Language Use and Cognition Two tasks each to assess linguistic ability (lexical decision making and word recognition), monitoring (squares and digits), and working memory list-switching (figural and numerical). Oschwald, Schättin, von Bastian, & Souza (2018)
Multifacet Working Memory Testing and Training Package Two versions of the Tower of Fame task (Complex Span and Switching tasks coming soon), one for testing baseline performance and one for training. von Bastian, Langer, Jäncke, & Oberauer (2013)
Shifting Four tasks for each of the five shifting components rule, dimension, mapping, response set, and stimulus set switching. Each set of tasks comprises two two-choice, one three-choice, and one four-choice task. von Bastian & Druey (2017)
