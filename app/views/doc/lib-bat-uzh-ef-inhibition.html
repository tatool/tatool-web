Executive Functions (UZH-EF)

Inhibition

This set of tasks measures the inhibition of predominant responses. In the Simon task, a red or green circle is presented on either side of the screen. The task is to press the left arrow key when the circle is green and the right arrow key when the circle is red while ignoring the spatial location of the circle. The spatial location in which the circle appears and the spatial location of the response can be either congruent (e.g., a green circle appearing on the left, 75% of the trials) or incongruent (e.g., a green circle appearing on the right, 25% of the trials).

In the Flanker task, a string of 7 characters is presented. The task is to press the left arrow key when the character in the middle is a vowel (A or E), and the right arrow key when that character is a consonant (S or T) while ignoring the three surrounding stimuli on each side. These flankers are either congruent (i.e., from the same category as the target, for example AAAEAAA), incongruent (i.e., from the other category than the target, for example AAATAAA), or neutral (i.e., unrelated to the target's category, for example ###S###). Congruent, incongruent, and neutral trials occur equally often.

In the Stroop task, 1 to 4 characters are displayed. The task is to indicate the number of characters presented while ignoring their value. In congruent trials, the number of characters corresponds to the digits displayed (e.g., 333), whereas in incongruent trials, the magnitude of the digits differs from their number (e.g., 44). Neutral trials were unrelated symbols (e.g., ###). All three trial types occur equally often.

Simon Task

Executable

Use the following settings in the Module Editor for these tasks:

Project tatool
Executable Type tatoolSimon

Properties

See Simon Task in the Task Library for a description of the Properties that can be provided in the Module Editor to change the behaviour of the task.

The following values for the Properties were used in the publication:

Property Name Value
stimuliFile See Stimuli Files below
stimuliPath Project: uzh-ef-battery
Type: stimuli
showKeys Yes
timerEnabled No
displayDuration N/A (not set)
displayCrossDuration 250
randomisation N/A (not set)
