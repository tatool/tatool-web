Analytics

The tab Analytics allows researchers to monitor their modules and to download the data collected. Researchers can select modules they previously added to their Repository via drop-down menu (1). While public modules are always anonymised by using an internal user codes, researchers can choose whether they want to run their experiment in blind mode (using codes as well) or whether they want to list users by email adress (2). For each individual, Session data can be inspected and downloaded in form of zipped CSV files (3). Once an experiment is over and all data is downloaded, the module can be deleted from Analytics (4). Note that this irreversible step deletes all data from the web server.
