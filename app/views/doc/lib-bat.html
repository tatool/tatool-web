Task Batteries

Task Batteries are shared by researchers for reuse by others. They contain ready-to-run modules including instructions and stimuli files alongside the Executables. On each battery's page, you'll find information on the background, information on how to cite the battery, and publications that report data using the battery.

The following Task Batteries are currently available:

Name Content Authors (Year)
Executive Functions Three tasks for each of the executive functions inhibition (Simon, Flanker, and Stroop task), shifting (Shape/Color, Animacy/Size, and Parity/Magnitude Switching task), and updating (Color, Letter, and Number Keep-Track task). von Bastian, Souza, & Gade (2016)
Language Use and Cognition Two tasks each to assess linguistic ability (lexical decision making and word recognition), monitoring (squares and digits), and working memory list-switching (figural and numerical). Oschwald, Schättin, von Bastian, & Souza (2018)
Multifacet Working Memory Testing and Training Package Two versions of the Tower of Fame task (Complex Span and Switching tasks coming soon), one for testing baseline performance and one for training. von Bastian, Langer, Jäncke, & Oberauer (2013)
Shifting Four tasks for each of the five shifting components judgment, dimension, mapping, response set, and stimulus set switching. Each set of tasks comprises two two-choice, one three-choice, and one four-choice task. von Bastian & Druey (2017)
