Language Use and Cognition (UZH-LuCo)

Monitoring

This set of tasks measures monitoring of elements in working memory with the monitoring paradigm modelled after Oberauer, Süß Wilhelm, & Wittman (2003) (see also von Bastian & Oberauer, 2013; von Bastian, Souza, & Gade, 2016). The task is to monitor independently changing objects and react whenever a predefined relateion between the objects occurred. In the Squares Monitoring Task, dots are changing their position in a 10x10 grid every 2s. The task is to press the spacebar whenever four dots formed a square. In the Digits Monitoring Task, three-digit numbers are presented in a 3x3 grid, one of which changes every 2s. The task is to press the spacebar whenever the last digit of three numbers in a row, column or on a diagonal was identical.
